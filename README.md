# pblproject.github.io
PBL - PROJECT
This is a simple website which will calculate the eligibilty of a selected student. Pre-defined criteria for eligibilty of every student 
is specified depending on the year the student is currently studying in.

Students can acces their respective student portal to check their eligibility. Depending on the the number of subjects passed, eligibility
is calculated and displayed.
Admin portal can be accessed by the staff involved in adding and updating the results of the students. Admin can also check the eligibility
of each student.
